# Terrasul Terraplanagem

Site institucional desenvolvido para apresentar os serviços, maquinários e formas de contato da empresa **Terrasul Terraplanagem**.

A Terrasul atua no setor de **terraplanagem, locação de máquinas e transporte**, oferecendo soluções para obras, infraestrutura e serviços de movimentação de solo.

---

## Sobre o projeto

Este projeto consiste em um **site institucional estático**, desenvolvido com tecnologias web básicas para garantir rapidez, simplicidade e fácil manutenção.

O objetivo do site é:

* Apresentar a empresa
* Mostrar os maquinários disponíveis
* Divulgar os serviços prestados
* Facilitar o contato com clientes

---

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages (hospedagem)

## Funcionalidades

* Apresentação institucional da empresa
* Seção de maquinário
* Galeria de imagens
* Links para contato
* Integração com WhatsApp
* Layout responsivo

---

## Objetivo do site

O site foi desenvolvido para fortalecer a presença digital da empresa e facilitar o acesso de clientes aos serviços oferecidos pela Terrasul.
